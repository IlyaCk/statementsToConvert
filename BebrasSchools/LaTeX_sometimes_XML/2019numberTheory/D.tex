\begin{problemAllDefault}{Перевірка на простот\'{у}--3}

Задача відрізняється від двох попередніх \emph{лише} обмеженнями:

\vspace{-0.5\baselineskip}

\begin{multicols}{3}
\begin{itemize}
\item
$1\<A$;
\item
$B\<10^{10}$;
\item
$A\<B\<A+10^5$.
\end{itemize}
\end{multicols}

\end{problemAllDefault}
